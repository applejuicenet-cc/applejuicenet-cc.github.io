appleJuice Portable für Windows bestehend aus:

  • dem aktuellsten AdoptOpenJDK JRE 1.8
  • dem aktuellsten Core
  • dem aktuellsten JavaGUI

Features:

  • der appleJuice Client kann auf einem separaten Datenträger betrieben werden
  • keine Reste, oder Integration im Betriebssystem
  • Es muss kein JRE (Java) installiert werden
  • das ajfsp:// Protokoll kann bei Bedarf mit dem GUI verknüpft werden
  • der Arbeitsspeicher für den Core wird automatisch berechnet (-Xmx)
