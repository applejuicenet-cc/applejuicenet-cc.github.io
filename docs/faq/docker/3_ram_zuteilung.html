3. Wie grenze ich RAM des Core im Container ein?

Der RAM des Cores im Container kann ausschließlich beim Erstellen des Containers wie folgt eingeschränkt werden: --memory="2GB" (Beispiel für 2 GB RAM), siehe auch docker.com

Damit das wirklich funktioniert, müssen unter Linux zwingend die cgroups aktiviert sein.

Die cgroups sind bei den meisten Linux-Distributionen standardmäßig aktiviert.

Ob die cgroups aktiviert sind, kann mit folgendem Befehl überprüft werden:

cat /proc/cgroups (Wenn eine Liste erscheint, sind die cgroups aktiviert)

cgroups erklärt auf dev-insider.de
Auf einem Raspberry PI muss die Datei /boot/cmdline.txt angepasst werden
