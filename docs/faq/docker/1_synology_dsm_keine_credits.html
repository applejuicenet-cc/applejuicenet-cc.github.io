1. keine Credits unter Synology DSM

Die Docker Implementierung im Synology DSM hat einen Fehler im Netzwerk Stack.
Der Container muss zwingend im Host-Network laufen.
Beim erstellen des Containers die "Erweiterte Einstellungen" öffnen und im Reiter "Port weiterleitung" unten das Häkchen dafür setzen.

Ein bereits vorhandener Container kann nicht umkonfiguriert werden.
