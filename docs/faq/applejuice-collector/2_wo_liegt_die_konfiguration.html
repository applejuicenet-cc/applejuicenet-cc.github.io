2. Wo liegt die Konfiguration?

Die Konfiguration des Collectors besteht lediglich aus der
core-information-collector.xml Datei und liegt an folgendem Ort:

Windows C:\Benutzer\%USERNAME%\appleJuice\
Linux ~/appleJuice/ bzw. /home/$USER/appleJuice/
macOS ~/appleJuice/ bzw. /Users/$USER/appleJuice/
Docker im Container Volume /app/appleJuice/collector/
