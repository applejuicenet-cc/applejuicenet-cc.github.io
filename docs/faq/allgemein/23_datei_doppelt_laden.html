23. Kann ich eine Datei erneut herunterladen?

  • Situation: Ich möchte eine Datei erneut downloaden
  • Problem: Der Link bzw die Datei wird augenscheinlich nicht vom Core übernommen
  • Ursache: Die Datei wurde schon mal geladen, dann gelöscht oder verschoben
  • Lösung: Der Core muss neu gestartet werden damit er die vorherige Datei vergisst
