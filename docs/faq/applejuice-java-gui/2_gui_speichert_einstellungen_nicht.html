2. JavaGUI speichert die Einstellungen nicht (mehr)

Ursache:
Das JavaGUI wurde "als Admin" gestartet. Dadurch wurde die Konfiguration der JavaGUI mit erhöhten Rechten abgespeichert.

Öffnet man die GUI danach wieder als "normaler User", kann die Datei nicht mehr beschrieben werden.

Lösung:

  • beende das JavaGUI (z.B. über den TaskManager)
  • öffne den Ordner C:\Benutzer\%USERNAME%\appleJuice\gui\
    • unter macOS und Linux ~/appleJuice/gui/
  • entferne die Datei ajgui.properties
  • starte das JavaGUI erneut
