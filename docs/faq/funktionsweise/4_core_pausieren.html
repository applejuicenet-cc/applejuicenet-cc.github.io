4. appleJuice Core pausieren

Der Core kann nicht einfach so pausiert werden.

Unter linux kann der Prozess selbst "eingefroren" werden. z.B. mit killall -s STOP java und "aufgetaut" wird dann z.B. killall -s CONT java

Ansonsten reicht es evtl. auch, alle Downloads zu pausieren und den Upload auf 3 KiB/s (ist der Minimalwert) zu reduzieren.
